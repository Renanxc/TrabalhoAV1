Before hudson, Now jenkins

Posted by Yuri Lima on August 24, 2017

O Jenkins é um programa baseado em Java para auxiliar na integração contínua. Ele é uma versão forkada do Hudson. Com ele podemos buildar projetos, realizar o deploy dos mesmos e o próprio jenkins informa um log ao final do build.

Para configurar o Jenkins basta descompacta-lo no path do seu servidor de aplicação.

O Jenkins oferece a opção de enviar e-mail para a equipe caso tenha ocorrido algum problema ao tentar buildar, código possua algum erro ou a alteração que você tenha feito desconfigure algum módulo do seu projeto.
